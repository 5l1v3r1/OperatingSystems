Click File from left-top corner
Select New->Project
Choose Empty Project
Name it as Parent

In Solution Explorer window,under Parent project right click Source Files
Select add->new item
Chose C++ file
Name it as Parent.c

Right clik Solution 'Parent'
Choose add-> new project
Choose Empty Project
Name it as Child

In Solution Explorer window,under Parent project right click Source Files
Select add->new item
Chose C++ file
Name it as Parent.c
Copy-paste Parent.c to new Parent.c file

In Solution Explorer window,under Child project right click Source Files
Select add->new item
Chose C++ file
Name it as Child.c
Copy-paste Child.c to new Child.c file

In Solution Explorer window right click Parent project and lick Set as startup project
Build Solution and run it.
