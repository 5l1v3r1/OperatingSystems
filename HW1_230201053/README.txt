Click File from left-top corner
Select New->Project
Choose Empty Project
Name it as Parent
In Solution Explorer window,under Parent project right click Source Files
Select add->new item
Chose C++ file
Name it as Parent.c
Right clik Solution 'Parent'
Choose add-> new project
Choose Empty Project
Name it as Child
In Solution Explorer window,under Child project right click Source Files
Select add->new item
Chose C++ file
Name it as childp.c
Copy-paste Parent to new parentp.c file
Copy-paste Child to new childp.c file
In Solution Explorer window right click Parent project
click Set as startup project
In Solution Explorer window right click Solution 
Select Open Folder in File Expolere
Put market.txt in Parent file.
Build Solution and run it.
